E10 - Visualization Options¶

Besides showing pysofi-preocessed images directly with matplotlib.pyplot, we also incorporate Bokeh for interactive visualizations. Moreover, after extracting emitters' photo-physical properties with MOCA, the results can be demonstrated with a transparency map.

In this notebook, we show different Bokeh-based visualization options. Users can choose one or multiple options based on their needs.

1. Interactive visualization with Bokeh.¶

With Bokeh, it is easy to zoom in/out, check pixels values and change field-of-view on images. pysofi implements three ways to plot images with Bokeh,

  1. bokeh_visualization for a single greyscale image
  2. bokeh_visualization_mult for plotting and comparing multiple greyscale images side-by-side
  3. bokeh_visualization_rgba for a single RGBA image

We recommend the user to choose different methods for different scenarios. For all three functions, we can set the following parameters:

  • image: a greyscale or RGBA image needs to be plotted. A list of images can be passed to bokeh_visualization_mult.
  • palette: choice of palettes. For a complete palette list in bokeh documentation, see here.
  • save_option: whether to save the Bokeh image as a .html file.
  • imshow_same: whether to plot the image in Bokeh with the same axis direction as matplotlib.pyplot.imshow. The default y axis direction in Bokeh is flipped comparing to imshow.
  • title_lst: a list of image titles. This is sprecial for bokeh_visualization_mult.

Note that when the image is too large (> 1500 * 1500 pixels), it is likely for users to come across an error using Bokeh. In this case, we recommend to use matplotlib.pyplot.imshow instead.

2. Constrast adjustment¶

To better visualize images from fluorescence microscopy (especially live-cell results), the user has the option to adjust display contrast using enhance_contrast(im, display_contrast). The image with enhanced contrast can be plotted with bokeh_visualization_rgba or saved in a .png/.avi file.

In [13]:
en_im = v.enhance_contrast(ave_im, display_contrast = 1.4)
v.bokeh_visualization_rgba(en_im)

3. Transparency map¶

After extracting emitters' photo-physical information (e.g., on-time ratio, brightness) using MOCA, we recommend the user to color-code the features and adjust the intensity information with a transparency map. For instance, on-time ratio profile can be color-coded with 'pink' colormap, and multiplied by the second-order moment reconstruction as the transparency map.

In pysofi, add_transmap(im, trans_map, cmap) can come in handy for this purpose.
