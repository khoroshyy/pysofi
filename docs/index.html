pysofi - SOFI 2.0 Analysis in Python¶

Pysofi is an open-source python pacakge for the super-solution image processing method SOFI. Pysofi supports tiff files with image stacks. Data can be acquired from cameras like EMCCD or sCMOS, or simulations.

We provide well-tested implementaions of SOFI2.0 on live-cell imaging data in Jupyter notebook, as well as demos and examples on each step of the processing.
